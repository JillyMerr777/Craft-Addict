# Craft Addict — basic demo

Simple static demo with a mid-century / 70s inspired aesthetic. The site will act as inspiration for crafty folks looking to start something new or share a project they've created already. Files:

- `index.html` — main page
- `styles.css` — styling
- `script.js` — client interactions (parallax, wizard, inspiration)

To open: just open `index.html` in a browser (no server required for basic functionality).
